# Photo Gallery App

This is a React Photo Gallery application built using React, Vite, and Tailwind CSS.

Features:
- Fetches photos from the Picsum API
- Responsive grid layout
- Search filter by author
- Mark photos as favourites
- Favourites persist using localStorage
- Uses React hooks such as useReducer, useMemo, and useCallback

Tech Stack:
- React
- Vite
- Tailwind CSS
